Easily creates image sprites. It will generate the sprite image and its CSS in no time, and you can add images later and regenerate it without having to worry about breaking your code! Key features: - All image files on the folder will be added to the sprite no matter the size. - Jpg, jpeg, png, bmp and gif extensions are supported. - The result 3 files (*.png, *.css and *.html). - Demo HTML auto-generated. No installation is required, just download, unzip and run. Requires Microsoft .Net Framework 4.0.
